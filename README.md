# mybatis-dtos-generator

> Springboot mybatis base project

## Installation

First, install [Yeoman](http://yeoman.io) (we assume you have pre-installed [node.js](https://nodejs.org/)).

```bash
npm install -g yo
npm link
```

Then generate your new project:

```bash
yo mybatis-dtos:entity
```
